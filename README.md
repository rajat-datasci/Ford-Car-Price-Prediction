\# Ford Car Price Prediction



\## About

ML model to predict Ford car

prices using Linear Regression.

Dataset: Kaggle (Adhurim Quku)



\## Steps

1\. EDA \& Data Cleaning

2\. Feature Engineering

3\. Encoding \& Scaling

4\. Model Training

5\. Evaluation (R², Adjusted R²)



\## Results

Train R² : 0.8495

Test  R² : 0.8485

Train Adjusted R² : 0.8492

Test Adjusted R²  : 0.8472

Pandas, NumPy, Scipy, Warnings
